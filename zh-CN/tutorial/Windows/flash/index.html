在浏览器中使用 Flash

发表于 2022-04-30 更新于 2023-01-05 分类于 教程 ， Windows
用 CleanFlash 和 Internet Explorer（模式）在 Windows 中游玩 Flash 网页游戏

用前说明

CleanFlash 是基于中国特供 Flash 简化而成，无广告后门，现已从 GitHub 移除
目前仍依旧在 Gitlab 中更新

实验环境

Microsoft Edge
（如果你不想用 Microsoft Edge 和 Internet Explorer 的话，请自行搜索“浏览器名字+怎么开启 IE 模式”）
CleanFlash

安装 CleanFlash

打开安装程序
勾选复选框，点击“AGREE”

然后这三个复选框默认勾选即可，点击“NEXT”
)
这里是安装 Flash 独立播放器，如果你用不到，把第一个复选框取消即可，然后点击“NEXT”

Flash 独立播放器是这样的

在这两个页面点击“NEXT”和“INSTALL”继续下一步即可


受网速影响，可能会安装较慢，过程中浏览器会被关闭
（部分安装程序在运行时可能会与 CleanFlash 安装程序冲突，请关闭后再尝试安装）
完成后点击“QUIT”即可

使用 CleanFlash

访问一个需要 Flash 的网站，然后开启 Internet Explorer 模式
（你也可以直接打开 Internet Explorer 来游玩，如果不会开启模式可以看 这个文章（Windows 10 也可以用文章中的方法1））

打开后网页会重新加载，等待游戏加载完成就可以了
