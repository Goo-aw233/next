发表于 2022-05-28 更新于 2022-08-21 分类于 教程 ， Windows

用前说明

仅支持 1507、1607、1809、21H2 版本的 Windows，对应的是 LTSB 2015、LTSB 2016、LTSC 2019 和 LTSC 2021
如果你需要转换到你想要的版本则需要清除数据降级/保留数据升级到对应版本
