安装 AFC2 和 AppSync

发表于 2022-01-01 更新于 2023-04-07 分类于 教程 ， iOS ， 越狱
给 iOS 安装 Apple File Conduit "2" 和 AppSync 插件

用前说明

实验环境（其他环境一样可以）

iOS6.1.3
Cydia

第三方非作者源

安装 AppSync

简介

AppSync 又名 AppSync Unified,是给越狱的设备安装越狱软件用的插件

如何安装

在 Cydia 中添加软件源搜索“AppSync”
下载名为”AppSync Unified”的插件即可

安装完后点击<font size=5 color=red“重启 SpringBoard”

安装 Apple File Conduit “2”

简介

Apple File Conduit “2” 又名 AFC2,是给越狱的设备浏览设备目录的插件

如何安装

在 Cydia 中直接搜索“Apple File”即可跳出相对应的结果

安装完后点击“重启 SpringBoard”
