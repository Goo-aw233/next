注册美区 Apple ID

发表于 2021-11-21 更新于 2023-04-07 分类于 教程
使用虚拟身份信息来面挂代理注册美区 Apple ID

简介

只需要虚拟身份就可以免代理注册美区 Apple ID

创建 Apple ID

创建 Apple ID
选择如下图的位置的选项(图示为英文)

并填写相对应的信息(年龄需要>13 建议18+ 出生日期要保存好)
TIPS：国家需要选择美国 电话号码选择+86或自己想用的电话号码即可

验证账户

登录 Apple ID 后点击验证账户

点击下一步 并打开同意条款与条件再点击下一页

然后填入该网站的虚拟信息(请不要用于任何非法用途)
(如果以上网页打不开可尝试这个 https://www.meiguodizhi.com)
TIPS:我的个人建议是在右边生成免税州的地址(如阿拉斯加)

生成后将对应的信息填入

以下是我个人填写的虚拟信息

信用卡/借记卡: 号码:4539185730103786 有效期:07/2025 安全码:524

名:Michael
姓:Numbers

街道:4641 Kidd Avenue
城市:Minto
州:阿拉斯加州(英文为 Alaska)
邮政编码:99758
电话:907 7988126

完成后再次登录 Apple ID 就可以成功使用了

为 Apple ID 充值

在淘宝、京东等店铺购买美区的 Apple ID 充值卡

点击 App Store 右上角的头像 – 兑换礼品卡或代码
输入店家提供给你的卡密即可兑换

Workstation 使用 PE 装系统
安装 AFC2 和 AppSync
