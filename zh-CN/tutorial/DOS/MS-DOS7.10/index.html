安装 MS-DOS 7.10

发表于 2022-05-08 更新于 2023-04-07 分类于 教程 ， DOS
使用软盘，给 Workstation 虚拟机安装 MS-DOS 7.10

用前说明

实验环境

MS-DOS 7.10 CD（选择 Server 2 或 Server 1 都可以
VMware Workstation（其它虚拟机也可以，部分内容有所不同）

创建虚拟机

选择系统版本中选择“其它”>“MS-DOS”

CPU 1p1c，RAM 64MiB 即可

硬盘选择“储存为单文件”，大小 2-4GiB 即可

然后开启虚拟机

安装 MS-DOS 7.1

在该页面输入“1”然后回车

在接下来的几个页面选择“Next”即可



在这里选择“Continue”

选择”Create a FAT32/16/12 Primary Partition”

完成后点击“Rebot now”

在这个页面的时候选择虚拟机关机

关机后选择“虚拟机”>“电源”>“开机时进入固件”

进入 BIOS 后用 → 选择到“Boot”，再将光标移动到“CD-ROM Drive”上
按下 + 使得该选项向上移动，然后按下 F10 选择“Yes”保存重启

在重启后该页面输入“1”然后回车

在接下来的几个页面选择“Next”即可



在这里选择“Skip”

点击“Yes”

默认路径即可，继续选择“Next”

继续“Yes”

在这里把“Install Add-Ons”前面的“x”给去掉

然后继续“Next”

如果你不想安装“AccessDos”可以选择“No”

点击“OK”

点击“Yes”

这个是启动日志，你不想要可以选择“No”

开启长文件名支持，可以选择“Yes”

默认的“Enable UMB memory”即可

你不确定要选什么的话就选择“Load both”

第二个是电源管理，第五个不知道，第六个是加快硬盘读写，我选择默认，然后继续“Continue”

代码页，选择倒数第二个后点“Continue”

终于结束了（喜），点击“OK”，然后点击“Yes”来重启系统

重启之后输入“1”（默认输入）即可启动 MS-DOS

这样就成了
