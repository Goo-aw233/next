屏蔽获取 Windows 11 更新

发表于 2022-10-30 更新于 2023-04-07 分类于 教程 ， Windows
使用组策略/注册表功能来阻止 Windows 自动更新到 Windows 11

用前说明

如果以下操作都没有办法屏蔽，可以再试试这个方法 屏蔽 Windows 各种更新

使用组策略

（家庭版没有组策略功能，所以需要用注册表，请浏览下一章
按下 Windows 徽标键+R 来打开运行，并输入 gpedit.msc 并回车

然后找到“计算机配置”>“管理模板”>“Windows 组件”>“Windows 更新”>“适用于企业的 Windows 更新”>“选择目标功能更新版本”

然后点击“已启用(E)”，然后在下面的第一个矩形框中输入 Windows 10，第二个框根据你的系统版本填写，如 22H2，最后点确定保存
但如果设置到了不再支持设置的版本，将强制执行功能升级

最后在“Windows 更新”里就会有提示了

Windows 11 更新的提示也没了

使用注册表

# Windows
Edge 开启预览视觉效果
屏蔽 Windows 各种更新
