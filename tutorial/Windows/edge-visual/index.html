Edge 开启预览视觉效果

发表于 2022-10-29 更新于 2023-04-07 分类于 教程 ， Windows
在新版 Microsoft Edge 中开启 Windows 11 视觉效果和圆角标签选项卡

用前说明

请确保你的 Microsoft Edge 版本 ≥102（最好是最新）

开启实验模式

在地址栏输入 edge://flags 并回车
在搜索栏中输入“Windows 11”并回车，找到以下几个项目并改为“Enabled”

然后再次在搜索栏输入“Rounded”并回车，找到项目并改为“Enabled”

最后点击下方的“重启”

重启 Microsoft Edge 后在地址栏输入 edge://settings/appearance 并回车
将这两个选项开启并重启 Microsoft Edge 后即可生效
红色矩形框是开启的功能，橙色矩形框是开启后的效果
