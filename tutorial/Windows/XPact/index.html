在安全模式下修复 Windows XP 没激活时无法进入桌面

进入安全模式

强制重启两次，在模式选择界面选择“安全模式”

然后选择“Administrator”（如果没有就选择自己的用户名）

点击“是”即可

重置激活

在安全模式下打开“开始菜单”，点击旁边的“运行(R)…”

然后在里面输入 cmd 并回车

然后在 cmd 里输入以下命令（区分大小写），然后回车（区分大小写）

完成后重启电脑

然后就可以成功进入系统了
不过会重置回30天后必须激活
