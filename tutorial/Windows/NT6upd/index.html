解决 Windows Vista 和 7 问题合集（1）

发表于 2022-05-16 更新于 2023-01-05 分类于 教程 ， Windows
给 Windows Vista/7 安装 Service Pack 和解决 Windows Vista/7/Server 2008 R2/Home Server 2011 无法获取更新

用前说明

实验环境

Windows Vista
Windows 7
Windows Server 2008 (R2)
Windows Home Server 2011
补丁
（章节 I 请选择 SP 文件夹内的内容，章节 II 请选择 Update Package 文件夹的内容，x64 为 64 位，x86 为 32 位）

章节 I（给 Windows Vista/7 安装 Service Pack）

为了节省资源，Vista 和 7 的教程合二为一，内容上大同小异
如果你不是虚拟机，则需要用虚拟光驱工具或者解压工具，并直接观看 安装补丁 章节

在虚拟机中插入补丁

点击虚拟机设置

在虚拟机设置中选择 CD/DVD，然后点击右边的使用 ISO 映像文件，并点击浏览，然后选择 ISO（这里就不放 Vista 的例图了）

选择完成后点击确认并返回到虚拟机

安装补丁

插入补丁后点击 Windows 徽标键>计算机

双击打开 CD 驱动器

直接运行镜像内提供的 exe 安装包即可

Windows Vista 则需要先安装 SP1 才可以安装 SP2

然后就完成了
