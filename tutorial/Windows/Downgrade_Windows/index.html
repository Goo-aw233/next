降级 Windows

发表于 2023-02-01 更新于 2023-04-07 分类于 教程 ， Windows
通过替换镜像内的文件使 setup 正常降级系统

替换 setuphost.exe 之前，Setup 无法正常保留文件降级系统

使用 UltraISO（“网站”——“资源列表”），将新版本镜像内 sources 文件夹下的 setuphost.exe 文件，替换到旧版本镜像，并保存

不知道镜像从哪下请看 使用 UUP 构建 Windows 镜像

如果无法保存
(没有截图，凑合看吧（)
将旧版本镜像内所有内容解压，点击 UltraISO 上方“启动”——“保存引导文件…”，随意存放到一个位置

将新版本镜像内 sources 文件夹下的 setuphost.exe 文件复制出来替换

用 UltraISO 新建镜像，将旧版本镜像内所有文件复制进去，点击 UltraISO 上方“启动”——“加载引导文件…”，并选择刚刚保存的引导文件，最后点击保存即可

替换完成后就可以保留文件降级 Windows 了


最后就可以正常使用（可能会出现一些奇奇怪怪的 Bug）
